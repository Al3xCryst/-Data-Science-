# -Data-Science-
Дипломная работа по курсу "Data Science: расширенный курс" от Нетологии

  Задачей моего дипломного проекта была бинарная классификация пневмонии на рентген-снимках грудной клетки с использованием сверточных нейронных сетей (ResNet18/50, DenseNet121) с учетом высокой чувствительности (recall ≥ 0.97) и минимазации ложных срабатываний False Positive.
  
*Метрики:*
  В качестве основных метрик использую recall, roc-auc, f1-score и confusion matrix(FP).
  Основной метрикой качества является метрика recall (полнота), тк она напрямую показывает, сколько реальных случаев пневмонии было найдено моделью.
  Метрика roc-auc показывает качество модели вне зависимости от порога, то есть насколько хорошо она сама ранжирует классы по вероятностям. Она важна в сравнении моделей между собой, потому что эта метрика не зависит от порога. (вычисляется как площадь под ROC – кривой - график, который показывает, как меняется соотношение между правильно найденными больными и ложными тревогами при изменении порога классификации).
    Метрика F1 – баланс между precision(точность положительного класса) и recall. При подборе порога(уменьшении) recall растет, а precision падает – повышается количество ложных срабатываний False Positive. F1 указывает, насколько хорошо модель балансирует между recall и precision. Один из основных критериев выбора модели, так как учитывает оба показателя одновременно.

*Анализ данных:*
  Использую датасет Chest X-Ray (Pneumonia) (5863 изображений, размеченных как NORMAL:1341 и PNEUMONIUA:3785) с kaggle. После анализа данных создал новый валидационный датасет(15% от train датасета), тк изначальная val выборка слишком мала (16 изображений). Так же,  из-за дибаланса классов, применяю неагрессивные аугментации к классу NORMAL в train датасете.
